## Description
**JPollen** is a library mainly aimed at reducing the volume of JSON with repeated keys and values, but in the near future I will try to add more features shown below in the roadmap.

## Library status

The library is not ready and I don't know when it will be useful in basic functionality.

## Roadmap

| Feature     | IsDone    |
|:--------|:---------:|
| Minimize JSON    | ❌   |
| Swap keys and values rule  | ❌   |
| Skip keys and values rule   | ❌   |
| Filter JSON with specific conditions   | ❌   |
| Operating on multiple contexts    | ❌   |
| Handling JSON streams   | ❌   |
| Search engine    | ❌   |
| Highlight searched items   | ❌   |
| Defining parameters / fields with extra rules | ❌   |
| Searching contents of files marked as file fields (Base64) | ❌   |
